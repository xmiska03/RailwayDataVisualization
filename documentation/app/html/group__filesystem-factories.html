const io::IOContext &  io_context,

Create a new FileSystem by URI with a custom IO context.

Recognized schemes are "file", "mock", "hdfs", "viewfs", "s3", "gs" and "gcs".

Support for other schemes can be added using RegisterFileSystemFactory.

Parameters
[in]uria URI-based path, ex: file:///some/local/path
[in]io_contextan IOContext which will be associated with the filesystem
[out]out_path(optional) Path inside the filesystem.
Returns
out_fs FileSystem instance.
