general_functions.calculate_train_profile_transformation_matrix (   trans_point,
  rot_mat_3x3 
)

Calculates a transformation matrix for train profile positioning.

It is needed to translate the frame to "trans_point" and then rotate around that point. That is equivalent to rotating around [0, 0, 0] and then translating to "trans_point".

Parameters
trans_pointThe position to which the profile needs to be translated.
rot_mat_3x3The rotation that needs to be applied to the profile.
Returns
The calculated transformation matrix.
