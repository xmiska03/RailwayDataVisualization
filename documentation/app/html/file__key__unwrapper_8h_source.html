32// This class will retrieve the key from "key metadata", following these steps:
33// 1. Parse "key metadata" (see structure in KeyMetadata class).
34// 2. Retrieve "key material" which can be stored inside or outside "key metadata".
35// 3. Unwrap the "data encryption key" from "key material". There are 2 modes:
36// 3.1. single wrapping: decrypt the wrapped "data encryption key" directly with "master
37// encryption key" 3.2. double wrapping: 2 steps: 3.2.1. "key encryption key" is decrypted
38// with "master encryption key" 3.2.2. "data encryption key" is decrypted with the above
39// "key encryption key"
