31// Functionality for converting Arrow data to Comma separated value text.
32// This library supports all primitive types that can be cast to a StringArray or
33// a LargeStringArray.
34// It applies to following formatting rules:
35// - For non-binary types no quotes surround values. Nulls are represented as the empty
36// string.
37// - For binary types all non-null data is quoted (and quotes within data are escaped
38// with an additional quote).
39// Null values are empty and unquoted.
