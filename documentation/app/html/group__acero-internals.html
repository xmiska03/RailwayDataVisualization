Detailed Description

A unit of work for kernel execution. It contains a collection of Array and Scalar values and an optional SelectionVector indicating that there is an unmaterialized filter that either must be materialized, or (if the kernel supports it) pushed down into the kernel implementation.

ExecBatch is semantically similar to RecordBatch in that in a SQL context it represents a collection of records, but constant "columns" are represented by Scalar values rather than having to be converted into arrays with repeated values.
