updateDeck()
Changes camera position. The final position of the virtual camera is a combination of: 1. the current position of the train (determined by the window.position variable, data is in arrays window.translations, window.rotations_inv, window.rotations_euler), 2. custom camera offset set by the user (window.camera_offset).
